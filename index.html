Daxchain builds practical Ethereum tooling and strange token experiments — five public projects are on the bench right now.

Daxie

An Ethereum wallet for AI agents, with local custody, JSON-first command-line workflows, a built-in MCP server, and spend guardrails an AI agent can't raise.

github.com/daxchain-io/daxie open repo -> PRE-ALPHA

Daxib

The Bitcoin sibling of Daxie — an AI-agent Bitcoin wallet with local custody, a built-in MCP server, and spend guardrails an AI agent can't raise. Currently in design.

github.com/daxchain-io/daxib open repo ->

EVM Tools

A suite of composable CLIs that stream EVM chain activity as newline-delimited JSON, then pipe it into any downstream system through a growing set of pluggable sinks.

github.com/daxchain-io/evm-tools open repo -> ERC-20

NZT-48

An experimental ERC-20 where every untouched address already shows a balance — a toy coin for poking at what wallets, indexers, and contracts assume.

github.com/daxchain-io/nzt48 open repo -> DEPRECATED

Blockchain Exporter

A Prometheus exporter that polls EVM chains and exposes their metrics. Now retired to light, AI-assisted maintenance — still here for existing setups, but for new work we'd point you to EVM Tools instead.

github.com/daxchain-io/blockchain-exporter open repo ->
