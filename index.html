Daxie CLI

An agent-first Ethereum wallet in Go: CLI, MCP server, local custody, JSON workflows, and spend-limit guardrails.

wallet
  • Non-interactive flags, environment variables, stdin secrets, and JSON output.
  • One wallet core underneath both the command line and the MCP server.
  • Operator-set policy limits sit below the agent-facing interface.

NZT-48

An experimental ERC20 where ordinary untouched addresses appear to start with 5,000 NZT-48.

erc20
  • Fresh accounts report a virtual balance before anything is materialized.
  • Transfers wake balances into real supply when accounts interact.
  • A toy coin for testing wallets, indexers, contracts, and human assumptions.
core trick
balanceOf(alice) -> 5,000 NZT-48
totalSupply()    -> 0 NZT-48
